Canvas
    Draw graphics on a webpage at runtime. The canvas only contains the graphics, and the whole drawing is done via JavaScript.
    Placeholders
    Provide a hint to what is to be inserted in a particular field of the form.
    Editable content
    Any element can be editable using some JavaScript event handler.
    Local storage
    Allows users to store data locally on their devices, which can be accessed even when the user is offline.
    The semantic header and footer
    The header and footer tags are new in HTML5. They allow it to be easier to define and find those parts of the layout.
    Email inputs
    Provides a range of in-built functionalities to help validate inputs like email addresses.
    Multimedia support
    Includes support for multimedia elements such as video and audio, which eliminates the need for additional plug-ins such as Flash.
